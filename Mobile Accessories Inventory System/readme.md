# Mobile Accessories Inventory System

An AI-powered chat-based inventory assistant that helps users check stock availability for mobile accessories using Airtable as the backend database.

## Features
- Responds to chat messages instantly  
- AI-driven replies using Gemini  
- Searches Airtable for product availability  
- Memory-enabled conversations for better context handling  

## How It Works
1. User sends a message asking about an accessory  
2. AI Agent interprets the query  
3. Airtable records are searched for stock details  
4. The bot returns availability, variants, and other info  

## Setup
Import workflow → Connect chat platform + Airtable → Add table IDs → Activate.
