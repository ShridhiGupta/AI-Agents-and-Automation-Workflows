# Chatbot for Course Selling Website

An AI-powered chatbot workflow that responds to website chat messages, fetches course details, answers FAQs, and logs leads automatically using Google Sheets.

## Features
- Real-time chat message handling  
- AI-based response generation (Gemini)  
- Course info lookup from Google Sheets  
- FAQ answering  
- Automatic lead entry into Sheets  
- Memory-enabled conversations for better context  

## How It Works
1. Chat message triggers the workflow  
2. AI Agent interprets the query  
3. Fetches course data or FAQs as needed  
4. Logs customer info to Sheets  
5. Sends an instant response back to the user  

## Setup
Import workflow → Connect chat platform + Google Sheets → Update sheet IDs → Activate.
