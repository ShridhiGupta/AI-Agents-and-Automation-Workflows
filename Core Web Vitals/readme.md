# Core Web Vitals Performance Tracker

An automated workflow to monitor **Core Web Vitals** for **Desktop and Mobile**, track historical performance, and send **email alerts** when metrics degrade.

## Features

* Scheduled Core Web Vitals checks
* Desktop & Mobile performance tracking
* Parallel execution and data merging
* Metric extraction and aggregation
* Historical data storage
* Automated email notifications

## Workflow

```
Schedule → Fetch Data → Process Metrics → Merge (Desktop & Mobile)
        → Store & Aggregate → Detect Issues → Send Email
```

## Use Cases

* Website performance monitoring
* SEO & UX optimization
* Performance regression detection
